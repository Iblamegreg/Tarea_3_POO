3.  **Calificación:** Procesamiento de promedios y determinación automática de estado (Aprobado/Reprobado).
4.  **Planilla de Luz:** Cálculos encadenados de subtotal, IVA (15%) y total de consumo.
5.  **Empleado:** Gestión de sueldos base con integración de bonos fijos.
6.  **Restaurante:** Sistema de pedidos con descuentos dinámicos según el monto consumido.
7.  **Inventario Profesional:** Sistema avanzado con protección de datos sensibles y reglas de negocio para control de stock.
